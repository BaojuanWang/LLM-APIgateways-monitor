# Model price results

This directory is generated by the Model Price Probe workflow.

Latest files (overwritten each run):

- `model_prices.csv`: model-level price records.
- `model_prices_summary.csv`: one-row-per-site status and quality summary.

Historical files (preserved for every run):

- `archive/model_prices_<UTC timestamp>.csv`
- `archive/model_prices_summary_<UTC timestamp>.csv`

The timestamp includes date and time, so scheduled and manual runs on the same day remain separate.
